Self-Checkout System (Clone of TESCO)

Technologies Used: Java, Visual Studio

Project Description

This project involved developing a self-checkout system that mimics the TESCO self-checkout interface using Java.

Key Features

  • Real-time item scanning
  • Total price calculation
  • Payment system simulation
  • User-friendly interface design
  • Robust error handling and user feedback mechanisms
