Self-Checkout System (Clone of TESCO)

Project Overview

This project involved developing a sophisticated self-checkout system that replicates the TESCO self-checkout interface, providing users with a seamless and efficient shopping experience.

Advanced item scanning system with instant recognition and price lookup

Payment System

Comprehensive payment system simulation with multiple payment methods
