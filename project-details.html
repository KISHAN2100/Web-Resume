Real-Time E-Commerce Android Application

Technologies Used: Android Studio, Firebase, Java, XML

Project Description

This project involved developing a fully functional e-commerce application for Android that allows users to browse products, manage their cart, and place orders in real-time.

Key Features

  • User authentication with Firebase
  • Real-time database integration for product management
  • Intuitive user interface designed with XML
  • Product search and cart management
  • Order placement functionality
